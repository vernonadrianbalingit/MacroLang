MacroLang

A domain-specific language for nutritional macro tracking

View on GitHub Download

About

MacroLang is a specialized programming language for tracking and calculating nutritional macros (protein, carbohydrates, and fats). It automatically calculates calories and can compare actual intake to personalized goals based on height, weight, and activity level.

Example

// Define foods with macros
food chicken_breast:
    protein 25
    carbs 0
    fats 3

food white_rice:
    protein 4
    carbs 45
    fats 0

// Personal goal definition
goal:
    height 180
    weight 80
    activity moderate
    objective bulk

Key Features

  • Define foods with protein, carbohydrate, and fat values
  • Use variables and arithmetic operations
  • Set personal goals for height, weight, and activity level
  • Get recommendations based on the Mifflin-St Jeor equation
  • Visual comparison of actual vs. recommended intake
  • FizzBuzz implementation based on protein intake

Installation

Dependencies

MacroLang requires Python 3.6+ and TextX.

pip install textx

Get MacroLang

git clone https://github.com/vernonadrianbalingit/MacroLang.git
cd MacroLang

Running MacroLang Programs

python interpreter.py examples/hello_macros.macro

Syntax

Food Declaration

food food_name:
    protein value
    carbs value
    fats value

Variables and Calculations

calculate variable_name = value
calculate double_protein = base_protein * 2

Goal Declaration

goal:
    height value
    weight value
    activity level    // sedentary, moderate, active
    objective target  // maintain, cut, bulk

Example Programs

The repository includes six example programs:

  1. hello_macros.macro - "Hello World" equivalent
  2. basic_tracking.macro - Simple food tracking
  3. variable_usage.macro - Using variables
  4. arithmetic_operations.macro - Arithmetic operations
  5. meal_planning.macro - Complex meal planning
  6. fizzbuzz.macro - FizzBuzz implementation

All example programs can be found in the examples folder of the GitHub repository: https://github.com/vernonadrianbalingit/MacroLang
